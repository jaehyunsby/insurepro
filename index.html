📖 사용법
마이서치 사용 가이드
올바른 순서로 설정하면 자동으로 연동됩니다
① 자산 등록 (가장 먼저)
가계부 → 자산 탭에서 보유한 자산을 모두 등록하세요.
• 계좌·체크: 급여통장, 생활비통장 등
• 신용카드: 주로 쓰는 카드
• 저축: 적금, 예금
• 보험: 납입 중인 보험 (회차 자동 관리)
• 부동산: 시세 + 대출금 함께 입력
• 대출·부채: 신용대출, 기타 대출
💡 자산을 먼저 등록해야 지출/이체 시 연동됩니다
② 정기관리 설정
가계부 → 정기관리 탭에서 매달 반복 항목을 등록하세요.

고정지출 등록
• 결제 수단에서 카드/계좌 선택
• 보험료면 '보험 연결' 드롭다운에서 해당 보험 자산 선택
• 매달 납부 버튼으로 실납부 처리 → 거래내역 자동 생성

저축 등록
• 출금 계좌 → 입금 자산 연결
• 완납 버튼으로 이체 자동 생성
• 보험 자산 연결 시 납입 회차 자동 증가
💡 납부 버튼을 눌러야만 자산 잔액에 반영됩니다
③ 일상 지출 입력
홈 또는 내역 탭의 + 버튼으로 입력하세요.
• 카드로 결제 → 자산에서 해당 카드 선택
• 계좌이체 → 해당 계좌 선택
• 계좌 간 이동 → 이체 선택
💡 자산을 연결해야 현금흐름 탭에서 흐름이 보입니다
④ 월말 카드 정산
가계부 → 자산 탭 → 💳 카드 정산
• 카드 선택 → 실제 청구서 금액 입력
• 출금 계좌 선택 → 정산 실행
• 계좌 → 카드 이체 자동 생성
• 차액은 기타지출 또는 카드혜택으로 자동 처리
💡 정산 후 카드 잔액이 0원이 됩니다
⚠️ 주의사항
• 카드 결제대금은 반드시 이체로 처리 (지출로 입력하면 이중계산)
• 대출 상환은 계좌 → 대출 자산 이체로 처리
• 부동산 대출 상환도 계좌 → 부동산 이체로 처리
• 고정지출 납부 버튼을 눌러야 내역에 반영됨
